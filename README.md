# devstar-mcp-server

devstar-mcp-server是DevStar Studio的组成部分，可以实现与大模型对话方式提交生成代码并提交pr等功能。

## 快速部署

* 本地部署DevStar代码托管平台
* 私有化部署大模型
* 部署Gitea MCP 服务器
* 配置Cursor启用MCP相关功能（可选配置指定公开的大模型或私有的大模型）

详细的[部署方法](docs/README.md)
