# DevStar MCP 使用示例

本文档提供了 DevStar MCP 服务器各种工具的详细使用示例和 prompt 模板。

## 🔐 用户管理

### get_my_user_info
获取已认证用户信息

**Prompt 示例：**
```
请帮我获取当前用户的详细信息
```

**使用场景：**
- 验证用户身份
- 获取用户配置信息
- 用户权限检查

### search_users
搜索用户

**Prompt 示例：**
```
请搜索用户名为 "john" 的用户，返回匹配的用户列表。
```

**使用场景：**
- 查找特定用户
- 用户管理
- 权限分配

## 📦 仓库管理

### create_repo
创建新仓库

**Prompt 示例：**
```
请帮我创建一个名为 "my-project" 的新仓库，设置为私有仓库，描述为 "这是一个示例项目"。
```

**使用场景：**
- 新项目初始化
- 代码托管
- 团队协作

### list_my_repos
列出用户拥有的仓库

**Prompt 示例：**
```
请列出我拥有的所有仓库，包括仓库名称、描述、创建时间和最后更新时间。
```

**使用场景：**
- 仓库概览
- 项目管理
- 存储空间检查

### search_repos
搜索仓库

**Prompt 示例：**
```
请搜索包含 "web" 关键词的仓库，返回匹配的仓库列表。
```

**使用场景：**
- 查找特定项目
- 代码复用
- 项目发现

请从 main 分支创建一个名为 "feature/user-auth" 的新分支。

**使用场景：**

**Prompt 示例：**

**使用场景：**
- 清理旧分支

### list_branches

**Prompt 示例：**

请列出 "my-project" 仓库的所有分支，包括分支名称、最新提交信息和最后更新时间。
```

**使用场景：**
- 分支概览
- 代码审查
- 版本追踪

## 🐛 问题管理

### create_issue
创建新问题

**Prompt 示例：**
```
请创建一个新的 issue，标题为 "修复登录页面样式问题"，描述为 "登录页面的按钮样式在移动端显示异常，需要调整响应式设计"。
```

**使用场景：**
- Bug 报告
- 功能请求
- 任务跟踪

### list_repo_issues
列出仓库问题

**Prompt 示例：**
```
请列出 "my-project" 仓库的所有 open 状态的 issue，按创建时间排序。
```

**使用场景：**
- 问题跟踪
- 项目管理
- 优先级排序

### create_issue_comment
创建问题评论

**Prompt 示例：**
```
请在 issue #123 下添加评论："这个问题我已经在本地复现，正在修复中，预计明天完成。"
```

**使用场景：**
- 问题更新
- 团队沟通
- 进度跟踪

### edit_issue_comment
编辑问题评论

**Prompt 示例：**
```
请将我在 issue #123 下的最新评论修改为："修复已完成，正在测试中。"
```

**使用场景：**
- 信息更正
- 评论更新
- 内容维护

```

**使用场景：**
- 代码合并
- 功能集成
- 代码审查

### list_repo_pull_requests
列出拉取请求

**Prompt 示例：**
```
请列出 "my-project" 仓库的所有 open 状态的 pull request，包括标题、作者和创建时间。
```

**使用场景：**
- PR 概览
- 代码审查
- 合并管理

## 💡 高级使用技巧

### 组合使用多个工具
```
请帮我完成以下任务：
1. 搜索包含 "api" 关键词的仓库
2. 在找到的仓库中创建一个新的 issue 报告文档缺失问题
3. 获取该仓库的 README 文件内容进行分析
```

### 批量操作
```
请列出我拥有的所有仓库，然后为每个仓库创建一个 "需要更新依赖" 的 issue。
```

### 自动化工作流
```
请帮我：
1. 从 main 分支创建新分支 "chore/update-deps"
2. 获取 package.json 文件内容
3. 分析依赖版本
4. 创建 issue 记录需要更新的依赖
```
